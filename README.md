# 订阅提醒 (Subscription Reminder)

基于 Electron 与原生桌面 Web 技术打造的一款极其轻量、极简唯美的 Windows 原生风格毛玻璃桌面小组件。

## ✨ 特色
- 🎈 **轻量化**：完全脱离浏览器的桌面挂件，支持鼠标自由拖拽。
- 🧊 **极致通透**：完美接入 Windows 11 原生 Acrylic 亚克力系统级毛玻璃背景，能透出桌面壁纸及底层窗口。
- 📍 **全局置顶**：提供一键固定在屏幕最顶层功能，工作/摸鱼两不误。
- 📂 **全本地化数据**：利用 `localStorage` 原地保存您的每一条隐私的到期数据，绝不联网上传。

## 🛠️ 本地运行
1. `npm install`
2. `npm start`

## 📦 打包发给朋友 (Portable 便携版)
运行 `npm run build`，即可在 `dist/` 目录下找到可直接双击运行的独立 `.exe` 文件！
